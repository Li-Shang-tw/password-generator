# password-generator

## 簡介
**以 vanilla Javascript做成的密碼生成器**
- 根據所選的長度以及字符元素來產生密碼
- 以客製化的滑動條選取長度,長度範圍為4~20
- 字符元素包括 4種選項: 1.大寫字母 2.小寫字母 3.數字 4.特殊字符
- 判斷密碼強度,顯示強度為 very weak、weak、medium、strong
- 生成密碼後,點選複製圖案即可將密碼複製到剪貼簿上
- 響應式設計
